Project Overview

This project is a full-stack e-commerce application developed during an internship. The backend handles product management, user registration/login, shopping cart functionality, and order processing. Built with Node.js, Express.js, and MongoDB, it provides RESTful APIs for integration with a frontend developed in HTML, CSS, and JavaScript.

Technologies Used

Backend: Node.js, Express.js

Database: MongoDB, Mongoose

Authentication: JWT (JSON Web Tokens)

Other Tools: Postman (for API testing), bcrypt (for password hashing), dotenv (for environment variables)

Features

User Management: Registration, login, password hashing, JWT authentication

Product Management: CRUD operations for products

Shopping Cart: Add, update, remove items

Order Processing: Place orders, track order history and status

Server-side Validation: Ensures data integrity and security

Fully integrated with frontend for a complete full-stack application experience
